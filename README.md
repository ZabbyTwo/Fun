# C++ Data Structures & Algorithms (DSA)

Welcome to my playground for learning, practicing, and studying Data Structures and Algorithms! This repository contains custom implementations of core computer science concepts written entirely in C++.

## Current Implementations

### Linked List
A custom singly linked list implementation featuring proper memory management and pointer traversal:
* **Insert:** Append new nodes containing integer data to the end of the list.
* **Remove:** Find and delete specific nodes by their value, safely freeing the memory.
* **Print/List:** Traverse the list and output the data alongside its next memory address (features German console output).

*(More algorithms like sorting coming soon!)*
